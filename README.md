## Features

### Scooter Shop view
 View list of shops
 Create a shop
 Read (list shops)
 Update a shop
 Delete a shop

### Public shop view
* Logged in users only (not!)
* See firstname, lastname, text and weblink
* add \[puddinq_admin_info\] as shortcode in posts / pages
